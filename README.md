# Geocash

## Summary
This hybrid mobile app was built during API World Hackathon 2017. It uses Ionic and AngularJS framework. 

There are two main functions of the app:
1. It alerts the user about his/her spending budget when the user enters a new location such as a restaurant or a shopping mall.
2. It notifies how much the user has actually spent when the user leaves the location. 

APIs Used: Citi FinTech API, HyperTrack API, Back4App API
